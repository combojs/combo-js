# Combo.render

Render a component inside a container element.

----------------------------------------------------------------------

## Usage

Combo.render(component, [data = {}])

### Params

| Param           | Type          | Details                       |
| --------------- | ------------- | ----------------------------- |
| component       | `Object`      | The component.                |
| data            | `*`           | The data.                     |
